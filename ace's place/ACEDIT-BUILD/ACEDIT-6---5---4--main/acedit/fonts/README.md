# ACEDIT Compiled Fonts

Output directory for compiled font binaries. Generated by Stage 5 (COMPILATION) of the build pipeline.

## Artifacts

| File | Format | Purpose | Typical Size |
|------|--------|---------|-------------|
| `ACEDIT-Regular.otf` | CFF OpenType | Primary format (cubic Bezier outlines) | ~46 KB |
| `ACEDIT-Regular.ttf` | TrueType | Fallback format (quadratic outlines) | ~270 KB |
| `ACEDIT-Regular.woff2` | WOFF2 | Web deployment (Brotli-compressed) | ~19 KB |

## Key Properties

- **Glyphs:** 1,097 (1,094 PUA + .notdef + space + nbspace)
- **Monospace:** All advance widths = 618 (UPM / phi)
- **UPM:** 1000
- **PUA range:** U+E000 through U+E445 (contiguous, zero gaps)
- **WOFF2 budget:** < 150 KB

## Regeneration

```bash
cd acedit
python3 scripts/build.py            # full pipeline
# or via the orchestration server:
python3 server.py                    # then POST /api/build
```

## Web Deployment

```css
@font-face {
  font-family: 'ACEDIT';
  src: url('fonts/ACEDIT-Regular.woff2') format('woff2'),
       url('fonts/ACEDIT-Regular.otf') format('opentype');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
  unicode-range: U+E000-F8FF;
}
```

ACEDIT extends JetBrains Mono -- PUA only. The `unicode-range` directive ensures the browser only downloads ACEDIT when PUA codepoints appear on the page.
