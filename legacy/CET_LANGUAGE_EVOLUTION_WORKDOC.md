## §10 GOVERNING PRINCIPLES

All edits must satisfy:

1. **No CET ontology.** CET vocabulary is used for narration and physical reading. CET's physical primitives are never imported as tier-zero foundations.
2. **No free parameters.** CET's free coefficients (α₁...α₆, ε_φ, ε_λ) are never imported.
3. **No reversed derivation.** Physics narrates the algebra; algebra derives the physics. CET's narration enriches the physical reading without overriding the derivation order.
4. **SIL compliance.** Every remark added gets an explicit SIL status. Interface Emergence at each level is FORCED (follows from {R,N}=N + tower lift). Geometric visualizations (helix, vortex) are RESONANT unless independently derived. Pressure/threshold narration is FORCED where it restates existing theorems in dynamical language, ENCODED where it adds terminology.
5. **Integration standard.** All additions read as if always present. No "we harvested from CET" language. No seams. The T_CREDIT entry handles attribution.
6. **Semantic discipline.** New vocabulary (interface, pressure, inheritance, residue/signature) follows the DICTIONARY's format: grid home, type, status, definition, tower ascent, cross-references, usage discipline.
